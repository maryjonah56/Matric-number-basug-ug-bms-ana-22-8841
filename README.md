## Overview of Git
Git is a distributed version control system used to track changes in files, particularly source code.  
It enables proper version management, collaboration, and recovery of previous file states using commits.

### Key Functions of Git:
- Tracks modifications made to files  
- Maintains complete version history  
- Supports collaborative development  
- Allows rollback to earlier versions  

---

## Overview of GitHub
GitHub is a cloud-based hosting platform for Git repositories.  
It provides a centralized environment for storing code, tracking changes, and sharing projects publicly or privately.

### Operational Process of GitHub:
- Creation of repositories  
- Addition of files to repositories  
- Saving updates through commits  
- Hosting repositories remotely for easy access and collaboration
